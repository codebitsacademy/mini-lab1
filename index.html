HTML Lists Reference

HTML Lists

Tag Name Description
< ol > Ordered List This tag list items out that will be marked with numbers by default
< ul > Unordered List This tag list items out that will be marked with bullets (small black circles) by default
< li > List Items This tag is used to represent an item in a list

HTML Lists Attributes

Attributes Name Description
type="1" Numbers The list items will be numbered with numbers (default)
type="A" UpperCase Letters The list items will be numbered with uppercase letters
type="I" Uppercase Roman Numbers The list items will be numbered with uppercase roman numbers

HTML Block Level Elements

Tag Name Description
< div > Division This tag defines a division or a section in an HTML document
< Form > Form This tag as a section of a document containing
normal content, markup, special elements called controls and labels on those controls
< p > Paragraph This tag defines a paragraph on an HTML document

HTML Inline Elements

Tag Name Description
< span > Span This tag a generic inline container for phrasing content,
which does not inherently represent anything. It can be used to group elements for styling purposes
< textarea > Textarea This tag defines a multi-line text input control.
A text area can hold an unlimited number of characters
< input > Input Field This tag specifies an input field where the user can enter data.

CSS Selectors

Tag Name Description
p{ color: red;} p Selector With is selector all < p > elements on the page will have a red text color.
#all_text{backgroung-color: green;} id selector With this CSS rule the HTML element with id="all_text" will have a background color of green.
.all_para{margin-top: 10px;} class selector With is CSS rule all HTML element with the class="all_para" will have a margin-top of 10pixels .

CSS Pseudo-classes

Tag Name Description
a:link {color: red;} unvisited link With this rule all unvisited < a > will have a color of red.
a:visited {color: brown;} visited link With this rule all visited < a > will have a color of brown.
a:hover {color:pink;} mouse over link With this rule all hovered < a > will have a color of pink.
a:active {color: blue;} selected link With this rule all active < a > will have a color of blue.
