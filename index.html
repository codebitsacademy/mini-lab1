HTML TABLE TAGS

Tag Name Description
<table> Table Head The set of rows defining the column headers in a table
<tbody> Table Body The set of row containing actual table data
<tr> Table Row The table row container
<td> Table Data The table row container
<tfoot> Table Foot The set of rows defining the footer in a table

Attributes Names Description

CSS PSEUDO-CLASSES

Class Description
:hover selects an element that is being hovered by a mouse pointer
:focus selects an element receiving focus from the keyboard or programattially
:active selects an element being clicked by a mouse pointer
:link selects all links that have not been clicked yet
:visited selects a link that has already been clicked

CSS CONVENTIONAL SELECTORS

Selector Name Description
head head tag selects the element with head tag
.red a defined class named red selects all elements with red class
#banky a defined id named banky selects all elements with Id called banky
div.row div tags with class named row selects all the element with div tag and 'row' class
