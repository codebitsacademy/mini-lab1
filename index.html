HTML Table Reference

Table Tags

Tag Name Description
table Table The wrapper element of all HTML tables.
thead Table Head The set of rows defining the column headers in the tables
tbody Table Body The set of rows containg actual table data
tr Table Row The table row container
td Table Body The table data container
tfoot Table Foot The set of rows defining the footer in a table
