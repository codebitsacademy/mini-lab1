Head Body Reference


Body Tags

Tag Name Description
<Body> Body It is like a house for all HTML documents and other tags outside the head tag
<Section> Section It is used to divide the contents in the HTML
<a> Anchor It is used to define the beginning and end of a hypertext link
<p> Paragraph It is used to designate paragraph in HTML


Form Tags

Tag Name Description
<form> Form It is used to create an HTML form for user inpute
<textarea> Text field It is a multi-line text input tag in HTML
<input> Input It specifies an input field where the user can enter data


Head Tags

Tag Name Description
<Head> Head It contains information relating to the documents
<title> Title It is an HTML element that specifies the title of a web page
<meta> Metadata It provides metadata about the HTML document
<link> Link It links between documents and external resource


CSS Positioning

Type Name Description
Static Positioning static Here element will stick to the normal page flow
Relative positioning relative Here element will adjusted relative to its normal position
Fixed positioning fixed Here element stays in one place even if the page is scrolled
Absolute positioning absolute Here elements are positioned relative to his first positioned ancestor
